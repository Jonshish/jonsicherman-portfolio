Ebay Cardscraper Project

( UNDER DEVELOPMENT )

About The Project

This eBay spider API project is in production to become a website dedicated to housing a price tracker/database of recently sold NFL quarterback rookie sports cards. As a sportscard collector myself, I find a lot of enjoyment working on this project. I wanted to challenge myself and create my own API instead of implementing the one eBay has to offer. Future plans are to create a Django Rest API out of the backend data, and present it via a React/Redux App on the frontend.



Technologies used - Python: Scrapy, Postgres SQL, Bootstrap, Javascript, ScrapingHub
